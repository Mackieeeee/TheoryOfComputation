Problem-1: Construction of a DFA for the set of string over {a, b} such that length of the string |w|=2 i.e, length of the string is exactly 2. Explanation – The desired language will be like:


L = {aa, ab, ba, bb}

The state transition diagram of the language will be like:



State A represents the set of all strings with length zero (0), State B represents the set of all strings with length one (1), and State C represents the set of all strings with length two (2). State C is the final state, while D is the dead state; this is because after receiving any letter as input, it will never enter the final state.


Number of states: n+2
Where n is |w|=n

The above automata will accept all strings with a length of exactly two. When the string length is 1, it will move from state A to state B. When the length of the string is 2, it will go from state B to C; if the length of the string is larger than 2, it will move from state C to D (dead state), and then from state D to D itself.

L = {aa, ab, ba, bb, aaa, aab, aba, abb........}

The state transition diagram of the language will be like:



Here, State A represent set of all string of length zero (0), state B represent set of all string of length one (1), and state C represent set of all string of length two (2).


Number of states: n+1
Where n is |w|>=n

The above automata will accept all strings with a length of at least two. When the string length is 1, it will move from state A to state B. When the string length is 2, it will go from state B to C, and when the string length is larger than 2, it will move from state C to C itself.
